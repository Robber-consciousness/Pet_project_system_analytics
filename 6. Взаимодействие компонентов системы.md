title Цифровая платформа управления данными и оборудованием промышленных комплексов

caption Обработка предиктивной и критической аварии\nMES / SCADA / IoT Platform

== Сбор телеметрии ==

loop Поток телеметрии в реальном времени

Sensors -> PLC : Передача телеметрии\n(температура, давление, вибрация)

PLC -> Gateway :Агрегация и первичная\nобработка данных

Gateway -> SCADA :REST/WebSocket\nПередача телеметрии

SCADA -> Alarm :Анализ пороговых значений\nи правил мониторинга

end

alt Обнаружена предиктивная авария

Alarm -> Kafka :Publish Event\nPredictiveAlertEvent

Kafka -> Integration :Consume PredictiveAlertEvent

Integration -> DB :Сохранение события\nи технических параметров

Integration --> MESAPI :REST API\nСоздание предупреждения

MESAPI -> MES :Регистрация инцидента\nв производственной системе

Integration -> Superset :Обновление аналитических\nдашбордов

Integration --> Notify :Инициировать уведомление

Notify --> Engineer :Уведомление инженеру АСУ ТП

Engineer -> MES :Анализ события\nи подтверждение обработки

MES -> DB :Обновление статуса события

else Обнаружена критическая авария

Alarm -> Kafka :Publish Event\nCriticalAlertEvent

Kafka -> Integration :Consume CriticalAlertEvent

Integration -> DB :Логирование аварии\nи параметров оборудования

Integration --> MESAPI :REST API\nСоздание Critical Incident

MESAPI -> MES :Регистрация критического инцидента

Integration --> Notify :Отправка критического уведомления

Notify --> Engineer :Push/SMS/Email\nКритическая авария

Notify --> Technologist :Push/SMS/Email\nОстановка технологического процесса

par Параллельные действия

Engineer -> MES :Создание заявки\nна обслуживание оборудования

Technologist -> MES :Подтверждение остановки\nтехнологического процесса

end

MES -> DB :Сохранение результатов\nреагирования

Integration -> Superset :Обновление KPI,\nаварий и аналитики

end

== Завершение обработки ==

MES -> DB :Фиксация итогового статуса\nинцидента

DB -> Superset :Обновление отчетности\nи аналитических витрин

Superset --> Engineer :Отображение статуса\nна Dashboard

@enduml
